# relatorio-vendaslucros-dio

## 📋 Sobre o Projeto
Projeto que tem como foco utilizar os recursos visuais do
PowerBI em benefício da análise de dados, facilitando a
compreensão do que está sendo abordado.

### 🔍 Desafios Encontrados
A questão visual e alguns elementos de agrupamento de dados
como um todo. Mas no geral, foi possível a compreensão clara
do objetivo do módulo.

### 🛠️ Tecnologias Utilizada
- **PowerBI**: gerenciando dados e visuais para composição do relatório.
